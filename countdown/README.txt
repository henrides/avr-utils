Countdown

PC3 - Decrement Button
PC4 - Increment Button
PC5 - Select Button

Move the cursor with the Increment & Decrement Buttons. Use select over a time unit to enter Edit mode.
With the Increment & Decrement buttons select a time. Press Select button to exit Edit mode.
Select "Start" to start the countdown. "Reset" put the timer back to 00:00:00.
